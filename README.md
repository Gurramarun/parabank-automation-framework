# ParaBank Automation Framework

This project is a Selenium WebDriver automation framework developed to test the functionality of the ParaBank web application. The framework is built using Java, TestNG, Maven, and follows the Page Object Model (POM) design pattern to ensure maintainable and scalable test automation.

## Features
- Automated user registration
- Login and logout validation
- Page Object Model implementation
- TestNG test execution and assertions
- Random test data generation for unique users
- Modular and reusable code structure

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Git & GitHub

## Project Structure
src/test/java
- base (WebDriver setup and teardown)
- pages (Page classes for UI interaction)
- tests (Test cases)
- utils (Utility classes like random data generator)

## Test Scenario Covered
1. Navigate to ParaBank registration page
2. Register a new user with unique credentials
3. Verify successful account creation
4. Log out from the application
5. Log in again using the registered credentials
6. Verify successful login

## Purpose
The goal of this project is to demonstrate the design and implementation of a scalable Selenium automation framework suitable for real-world testing environments.
